More information on the HighD dataset and how to obtain a copy can be found here: https://www.highd-dataset.com/.

Please use the following folder structure:

- data

    - HighD
        |

            |
            (place all data *.csv files here)
        |
        - images
            |

        |
        - README.txt
    |
    - Pneuma
    |
    - NGSim
